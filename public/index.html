Performer: Version 0.0.1 Pre-Alpha

My name is Blake West. I'm a developer and musician. I created this in 3 weeks as a tool to help you see how to play any piano song. The colored groups of blocks represent your left hand and right hand, respectively. All you need is a MIDI file (You can google a free one for almost any song), and the algorithm figures out the best fingers to use for each note, and then visualizes it. You can slow things down too!
It's certainly not perfect. The algorithm might choose wrong fingers for especially complicated songs. The 'hands' look funny. But you have to start somewhere. I hope you enjoy it!
